Sky localization uncertainty quantification for GWTC events

Results from processing LIGO/Virgo sky maps from the current (2024-08) Gravitational Wave Transient Catalog (GWTC) list of "confident" events; see GWTC.

The full catalog contains 93 events. The results here are for the 81 binary black hole (BBH) events that were fit with both the IMRPhenomXPHM and SEOBNRv4PHM waveform models (abbreviated as IMR and SEO in plot annotations). There are two plots for each event (on adjacent rows), one for each waveform model.

Each plot has two panels:

  • Left: Full-sky posterior PDF for the event location (Mollweide projection). Color indicates PDF value (integrated over a HEALPix pixel); lighter is higher density.
  • Right: Scalar uncertainty quantifications as textual annotations, giving expected search area (and its associated highest posterior density region (HPDR) level), information-equivalent range, areas of . Histogram (blue line) shows relative frequencies of areas searched until discovery in 2000 simulated perfect optimal searches of the posterior distribution; orange dots show corresponding values from an analytical calculation. Vertical line (brown) shows expected search effort (analytic); horizontal error bar (green) shows Monte Carlo estimate.
