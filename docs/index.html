This site hosts example results from use of the SkyMapUQ package for computing basic uncertainty quantifications summarizing posterior PDF sky maps reported as HEALPix maps.

The current example results are from preliminary calculations using data from the Gravitation Wave Open Science Center's Gravitational Wave Transient Catalog, specifically the list of 93 "confident" events from the LIGO/Virgo O1 to O3 observing runs. The calculations presented here used two maps for each of 81 GWTC events that have data from two waveform models, IMR and SEO.

Two collections of results are available:
  • Sky maps and uncertainty quantifications for each event (for both IMR and SEO fit results).
  • Scatterplots and histograms of uncertainty quantifications.
