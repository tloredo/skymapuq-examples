Sky localization uncertainty quantification for GWTC events

Results from processing LIGO/Virgo sky maps from the current (2024-08) Gravitational Wave Transient Catalog (GWTC) list of "confident" events; see GWTC.

The full catalog contains 93 events. The results here are for the 81 binary black hole (BBH) events that were fit with both the IMRPhenomXPHM and SEOBNRv4PHM waveform models (abbreviated as IMR and SEO in plot annotations).
